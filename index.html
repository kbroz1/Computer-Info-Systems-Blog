5 Components of a Computer System

Here are the 5 Basic Components of a Computer:

  1. CU-Control Unit:
    • Maintains order and directs operation of entire system
    • Acts as "Central Nervous System"
    • Directs and controls activities of internal and external devices

    • via GIPHY

  2. Input:
    • Information that is intereted intothe computer
    • Accepts/Reads list of instructions and data from outside world
    • Converts instructions/data in computer acceptable format
    • Supplies coverted instructions and data to computer system for further processing
    • Examples include keyboard, mouse, microphone

    • via GIPHY

  3. Output:
    • Links computer with external enviroment
    • Converts computer code (binary code) to human-readible format
    • Sends converted results to outside world
    • Examples include printer, monitor, projector, headphones

    • via GIPHY

  4. Storage Unit:
    • Stores programs, applications, data
    • Serves as computer's memory
    • Provides space for storing data and instructions, space for intermediate results and also space for the final results
    • Sends converted results to outside world
    • Examples include printer, monitor, projector, headphones
  5. ALU:
    • Stands for Arithmetic and Logic Unit
    • The place where instructions are executed during processing operations
    • All calculations an decisions are made in the ALU
